🎯 完整動作流程（cab 不透明，門機構清楚浮出）

    
1. 平層判定 — cab 到達 ±50mm → door zone sensor 觸發

    
2. vane 對位 — 轎門 vane（橘色片）進入 landing clutch（U 鉤）範圍

    
3. 馬達啟動 → 皮帶 → Crank Arm 推門吊架

    
4. 內門滑開 — vane 同步水平移動

    
5. 機械接合 — vane 物理夾入 clutch（U 鉤被夾住）

    
6. 內帶外開 — 內門繼續推 → vane 拉著 clutch → landing door 跟著滑

    

      為什麼 landing door 沒馬達也能開？

      機械「離合」設計：cab 平層時 vane 進入 clutch 才接合，cab 不在的樓層 vane 不在 → clutch 鎖住 → landing door 無法被拉。這就是電梯井道防墜落的關鍵！
