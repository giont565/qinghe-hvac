1客梯曳引機系統 — Traction Machine

永磁同步無齒輪曳引機（PMSM Gearless / MRL）為核心驅動器。馬達轉動 → 聯軸器 → 牽引輪 → 鋼絲繩 → 同時拉動轎廂上行 + 配重下行（或反向）。整段傳動為純摩擦力傳遞，無齒輪、無齒隙、效率 IE5、噪音 < 55 dB。
驅動
1.1 PMSM 永磁同步曳引馬達
型式PMSM Gearless（永磁同步無齒輪）
額定功率8.5 kW
額定轉速119 rpm（對應 cab 1.0 m/s）
額定扭矩580 Nm
效率等級IE5
位置回授Heidenhain EBI 1135 絕對編碼器（2¹⁹ × 4096，EnDat 2.2 介面）
散熱外加離心扇 450 m³/h（馬達 > 60°C 自啟）

動作原理

VFD 把 380 V 三相市電解碼成定頻定壓三相電→由內部 IGBT 模組逆變成可變頻、可變壓的 PWM 波形→驅動 PMSM 馬達定子線圈。永磁轉子被旋轉磁場拖動，產生扭矩。馬達跟皮帶輪（sheave）無齒輪直驅，所以馬達一圈 = sheave 一圈 = 鋼絲繩走 sheave 周長距離。

VFD 輸出 PWM → 定子三相線圈 → 旋轉磁場 → 永磁轉子被拖動 → 同軸 sheave 旋轉
機構
1.2 馬達軸 → 聯軸器 → Sheave 牽引輪

PMSM 屬於 gearless 直驅機構，馬達輸出軸與 sheave 共用同一根軸。彈性聯軸器負責吸收軸向偏差、避免馬達軸承受到鋼絲繩拉力造成早期損壞。

Sheave 直徑Φ400 mm
鋼絲繩槽數8 槽（每槽 V 形、繩槽角 35–40°）
摩擦係數0.10–0.12（鑄鐵繩槽 + 鋼絲繩）
包覆角≥ 180°（單繞 single wrap）
軸承SKF 雙列圓錐滾子軸承（永久封閉式潤滑）

力學原理

鋼絲繩跟 sheave 之間靠摩擦力傳遞動力。Sheave 旋轉時，cab 側鋼絲繩被往上拉，cw 側鋼絲繩同時被放下。鋼絲繩本身不滑動、繩槽內摩擦使繩跟著 sheave 一起跑。

安全
1.3 電磁制動器 Brake（雙鉗夾煞車碟）
型式Disc brake, fail-safe（失電抱閘）
釋放方式電磁線圈通電 → 推開鉗
抱閘方式斷電 → 彈簧推鉗夾碟
靜止扭矩≥ 2 × 額定載重扭矩
雙迴路主 + 備兩組獨立鉗（任一組可單獨抱閘）
反饋caliper 位置 sensor → 控制盤確認真實 release / set 狀態

動作原理（看 3D 模型）

  1. 通電釋放：控制盤確認所有安全條件 → 通電給線圈 → 線圈電磁力推開鉗 → caliper + 剎車片往外推 35 mm → 碟可自由旋轉、馬達才能轉動
  2. 運行中：線圈持續通電、保持鉗張開狀態（耗電 ≈ 30 W）
  3. 失電抱閘：任何安全故障 / 停機指令 → 切斷線圈電源 → 彈簧瞬間推鉗夾碟 → 摩擦阻止 sheave 旋轉 → cab 被鋼絲繩拉住停止
Fail-safe 安全特性 電磁失電型制動器是「Fail-safe」設計：失電 = 抱閘。因此任何電源故障、線圈斷線、保險絲熔斷、急停按下，都會立刻夾緊煞車碟。絕對不會出現「停電下墜」狀況。
機構
1.4 鋼絲繩 + 轎廂 + 配重 連動
鋼絲繩規格Φ10 mm × 8 條 / 8×19 Seale
安全係數≥ 12 : 1（破斷負荷 / 工作負荷）
cab 載重13 人 / 900 kg
cw 重量2,100 kg（≈ cab 自重 + 50% 額定載重）
速度1.0 m/s

配重設計原理

配重 = 轎廂自重 + 50% 額定載重。半載時馬達扭矩需求接近零（cab + 半載 ≈ cw、力矩平衡）。空載上行或滿載下行才需要全力扭矩。這樣馬達平均扭矩減半、能耗降低 40–60%，這就是為什麼電梯有配重。

空載上行：cab 輕 ↑、cw 重 ↓、cw 自己會往下拉 → 馬達只需「煞住別讓 cw 過快」+ 推 cab 上去 滿載下行：cab 重 ↓、cw 不夠重、cab 自己往下走 → 馬達也是「煞住」+ 推 cw 上去 平衡點（cab + 50% 載重）：馬達零扭矩、僅克服摩擦

2客梯門系統 — Door System

門系統包含「轎門 cab door」+ 4 個「層門 landing door」。所有層門皆無獨立馬達，被動跟隨 cab door — cab 對位後 vane（門刀）卡住 landing clutch（離合塊）一起拉開。整個 4 層樓只有一台 door operator（在 cab 頂）。
驅動
2.1 Door Operator 門機 — 馬達 → 聯軸器 → 主動 Pulley
馬達型式VFD 控 DC 馬達 / BLDC
聯軸器Jaw coupling 彈性爪型（SS 軸 + Hytrel 中間塊）
主動 pulleyΦ90 mm 鋁合金齒型輪
從動 pulleyΦ90 mm（位於門中央上方）
傳動皮帶封閉式齒型皮帶（HTD 8M）
開啟時間2.5 s（門距 800 mm）
保護photocell + 防夾邊扭矩限制

動作鏈

馬達轉一圈 → 聯軸器同步轉一圈 → 主動 pulley 轉一圈 → 齒型皮帶移動 πD = 282 mm → 從動 pulley 同步轉一圈
機構
2.2 皮帶夾 Belt Clamp + 連桿 Strut

齒型皮帶為封閉迴路：上排與下排走向永遠相反（pulley 順時針轉 → 上 belt 往右 + 下 belt 往左）。利用這個機械特性實現中分對開門：

  • 左門 的 belt clamp 夾在「下排」belt → 隨下 belt 往左移動
  • 右門 的 belt clamp 夾在「上排」belt → 隨上 belt 往右移動
  • 兩片門以相同速度反向打開、中間對開

連桿傳力

每個 belt clamp 透過垂直連桿 strut 接到 arm，arm 連到 cab door panel 的吊架 hanger，hanger 在 cab track 上滑動。皮帶移動 → clamp 移動 → strut 拉 arm → arm 拉 hanger → cab door 滑開。

機構
2.3 門刀 Vane + 門刀輪 Skate Rollers

cab door 左片外緣垂直方向裝有一塊鐵片，稱為門刀（vane / skate）。門刀末端裝有上下兩顆門刀輪（skate rollers），每顆 Φ40 mm 硬化鋼 + Delrin 外圈 + precision deep groove 軸承。

對位條件

  • cab 對位 landing 樓層水平誤差 ±50 mm 內
  • cab vane 跟 landing clutch 進入接合區（重疊距離 > 10 mm）
  • 滿足對位條件後、cab door 開門時、vane skate roller 才會頂到 landing 鎖機構
為什麼一定要對位才連動 如果 cab 沒對位、vane 跟 landing clutch 沒重疊：cab door 自己開、landing door 不動（鎖鉤還勾著）。這是故意設計的安全機制 — 萬一 cab 卡在樓層中間，乘客即便扳開 cab door 也無法打開 landing door 走出去（會掉井道）。
安全
2.4 層門 Clutch + 鎖鉤 + 電氣聯鎖

每個樓層門裝有完整的鎖機構（4 樓共 4 套）：

  • Clutch 離合塊：U 形夾鉤、跟 cab vane 接合用
  • Lock Hook 鎖鉤：機械鎖鉤、平常下垂勾住鎖體（lock body）
  • Lock Body 鎖體：被 hook 鉤住的承壓塊
  • Door Interlock Contact 電氣接點：NC contact, force-guided, 雙路、fail-safe
  • Interlock LED：綠 = 鎖緊允許運轉 / 紅閃 = 門開禁止

機械鎖原理

  1. 門關閉狀態：hook 下垂勾住 lock body → 即便外人扳門、機械鎖鉤頂住、無法打開
  2. cab 對位開門：cab vane skate roller 推 clutch → hook 被槓桿翻起 -π/4 → 釋放 lock body → 機械鎖解除 → vane 帶 clutch 一起拉、landing door 跟 cab door 同步打開
  3. 關門完成：cab door 跟 landing door 一起回到關門位置 → hook 重力下垂 → 重新勾住 lock body → 鎖緊

電氣聯鎖原理（雙重保護）

機械鎖之外、再加一層電氣 NC 接點：鎖鉤勾緊時、接點同步閉合、給主迴路通電允許運轉。任何一個樓層門被打開、對應的 NC contact 瞬斷、整個主迴路斷電、馬達立刻 STO（Safe Torque Off）、cab 緊急停車。

門關 + hook 勾鎖體 → NC contact 閉合 → 主迴路通電 → 允許電梯運轉 門開 (任意一層) → NC contact 瞬斷 → 主迴路斷電 → STO → 急停
EN 81-20 / CNS 17436 規範要求 門電氣聯鎖必須是「force-guided contact」、機械強制連動，不能用一般繼電器。任一接點熔接故障時，另一接點仍能正確切斷。雙路冗餘設計、SIL-3 等級認證。

3客梯安全防護鏈 — Safety Chain

電梯的安全是多重冗餘設計、不是「一道防線」。任何單一保護失效時、後續至少還有 2–3 道防線。整個 chain 任一節點斷開、安全繼電器（SIL-3）立刻切斷 VFD 使能、Brake 抱閘、Safety Gear 待命。
安全
3.1 限速器 Over-Speed Governor (OSG)
動作速度1.4 × 額定速度 = 1.4 m/s
機械動作離心式飛球 → 觸發 trip jaw → 夾住 governor rope
電氣訊號NC contact 切斷主迴路
重設方式技師手動 reset（不會自動恢復）

動作原理

  1. 正常運行：governor rope 跟著 cab 同步上下、governor wheel 旋轉、飛球離心力 < 彈簧復位力
  2. cab 超速 1.4×：governor wheel 轉速超過設定 → 飛球離心力 > 彈簧 → 飛球外擴觸發 trip jaw → 機械夾頭瞬間咬住 governor rope
  3. 同時兩件事：(a) governor rope 被卡住停止跟 cab 同步 → rope 拉力傳到 cab 上的 safety gear 楔塊 (b) governor NC 接點瞬斷主迴路
  4. 重設：技師到場、人工 reset trip jaw、檢查 governor 機構、檢查 safety gear、調查超速原因（VFD 故障 / 編碼器訊號丟失 / brake 卡住未抱閘）
安全
3.2 安全鉗 Safety Gear（楔塊式）
型式Type A Wedge (instantaneous 瞬時式)
數量cab 兩側各一組（左右 T 軌各一）
夾緊力≥ 6 × 額定載重
觸發方式governor rope 拉動 → 楔塊往 T 軌方向衝 60 mm
重設不可逆、需技師現場手動 reset

動作原理

cab 底兩側各有一個鋼性楔塊（wedge），平常隱藏在 cab 內。OSG 跳脫時、governor rope 被拉住、cab 繼續往下 → rope 相對 cab 往上、透過 cab 上的連桿機構往內推楔塊 → 楔塊衝出 60 mm、楔形咬入 T 型導軌兩側 → 摩擦力 > cab 重量 → cab 急停。

瞬時式 vs 漸進式 Type A 瞬時式用於低速電梯（≤ 0.63 m/s）。本案速度 1.0 m/s 應採用 Type B 漸進式（Progressive）：靠摩擦面逐漸增加咬合力、減速 g ≤ 1.0、降低乘客傷害。模型示意用瞬時式表示，實際施工須依規範升級。
安全
3.3 門電氣聯鎖 Door Interlock

請參考 §2.4 層門 Clutch + 鎖鉤 + 電氣聯鎖。重點：每個樓層門配備 force-guided NC contact、4 樓 × 2 路（主+備）= 8 個接點全串聯、任一斷則整個主迴路斷電。

安全
3.4 緩衝器 Buffer + 終端極限開關
cab 緩衝器聚胺酯油壓緩衝器 × 1（cab 正下方坑底）
cw 緩衝器聚胺酯緩衝器 × 1（cw 正下方坑底）
Top LimitHoneywell SZL Roller Lever、NC contact、頂端強制停止 — 比 final stop 早 5 cm 切斷上行電源
Bottom LimitHoneywell SZL Roller Lever、NC contact、底端強制停止 — 比 buffer 早 5 cm 切斷下行電源

三段保護

  1. 第一段（軟體）：VFD 接近終點減速、平層、停車
  2. 第二段（機械開關）：VFD 失效時、Top / Bottom Limit Switch 強制切斷馬達電源、brake 抱閘
  3. 第三段（緩衝器）：前面兩段都失效時、緩衝器吸收動能（最大可吸收 1.115 × 額定速度 = 1.115 m/s² 衝擊）
安全
3.5 門光電光幕 Photocell Curtain

cab door 兩側各裝一根紅外線光幕（光電 photocell + 紅外線發射）、垂直方向 64 道紅外線光束、保護範圍 0–1800 mm 高、5 ms 內偵測。任一光束被遮 → door operator 立刻反轉 → 門重新打開、等障礙物離開後再關。

關門中 → 任一光束被遮 → Photocell 觸發 → Door operator 反轉 → 門重新全開 → 等光束復原 + 3 秒延遲 → 重新關門

4客梯電控系統 — Electrical Control

電控系統集中在井道外的控制盤（ctrlElev）。櫃內依功能分層：主開關 / 變頻器 / CPU+HMI / I/O 模組 / 電源 + 接觸器 + 安全繼電器 / 端子排 / 散熱。每個訊號從感測器經 I/O 模組進 CPU、再由 CPU 輸出命令給 VFD 與接觸器。
電控
4.1 控制盤模組組成（從上到下 8 層）
第 1 層MCB 斷路器排（×5） + 主隔離開關手把 — 過載 / 短路保護、維修必先轉 OFF
第 2 層VFD KEB Combivert F6 11 kW + 散熱鰭片 + 藍 LCD — 主驅動器、內建 STO/SBC/SS1
第 3 層HMI 7" 觸控屏 — 故障歷史 / I/O 狀態 / 樓層位置、技師現場參數調整
第 4 層Motion CPU（32-bit ARM Cortex-M7、1ms 控制週期）+ 4 顆 LED（PWR/RUN/FAULT/COM）
第 5 層I/O 模組 ×6（DI8×2、DO8×2、AI4、AO2）— 32 個數位輸入 / 16 個數位輸出 / 4 個類比輸入 / 2 個類比輸出
第 6 層24V PSU + 主接觸器 K1/K2/K3 + Safety Relay SIL-3 — 整合 STO/SBC/SS1、急停 / OSG / 門接點任一斷則切斷 VFD 使能
第 7 層端子排 Terminal Blocks ×16 + 接線螺釘 — 外部 cable 進櫃接點
第 8 層散熱風扇 ×2（grille + 各 5 片扇葉、24VDC、櫃內 > 40°C 自啟）
電控
4.2 Sensor 訊號流
Top / Bottom LimitNC 接點 → DI 模組 → CPU 邏輯（極限觸發 → 接觸器斷電）
Encoder 編碼器EnDat 2.2 序列訊號 → VFD（位置反饋）+ CPU（樓層計算）
Load Cell 載重應變片電壓 → AI 模組 → CPU（過載判斷 / 補償扭矩計算）
Door Zone 樓層感應磁感應 + IR backup → DI 模組 → CPU（平層精度 ±3 mm）
Door InterlockNC 接點串聯 → 主迴路串聯（不經 CPU）→ Safety Relay → VFD 使能
OSG ContactNC 接點 → Safety Relay → VFD 使能 + DI 模組 → CPU 故障紀錄
Brake Sensorcaliper 位置感測 → DI 模組 → CPU（驗證真實 release / set 狀態、防止 brake 卡住）
Photocell光幕控制器 → DI 模組 → Door Operator CPU（門開合決策）
安全訊號 vs 控制訊號 安全訊號（門接點、OSG、急停、Limit）：直接串聯進主迴路、不經 CPU 處理、CPU 只「監看」紀錄事件。即使 CPU 死機也不影響安全動作。控制訊號（編碼器、載重、樓層）：進 CPU 處理後再輸出命令。設計上 SIL-3 安全功能完全獨立於 CPU。
時序
4.3 運行時序邏輯（以「1F → 3F 召喚」為例）
  1. Hall Call Registered：1F 大廳按上行按鈕 → DI 模組 → CPU 接收 → 燈號變綠
  2. Door Status Check：CPU 確認所有 4 樓 interlock 都閉合 + 急停未按 + OSG 未跳脫 → Safety Relay 通電允許 VFD 使能
  3. Brake Release：CPU 命令 → 制動器線圈通電 → caliper 張開 35 mm → brake sensor 確認真實 release → CPU 收到 OK 才下一步
  4. VFD Start：CPU 送速度曲線 (S-curve) 給 VFD → VFD 解碼為 PWM → PMSM 啟動 → 編碼器回授 → 閉迴路速度控制
  5. Cab Travel：cab 上行、每經樓層感應條 → CPU 更新樓層計算 → 接近 3F 時觸發減速曲線
  6. Door Zone Detected：3F 樓層感應條被 reader 觸發 → CPU 切換到平層模式 → 微速 0.05 m/s 對準
  7. Stop + Brake Set：平層精度 ±5 mm 內 → VFD 零速 → brake 通電狀態切換成失電 → caliper 抱閘 → 確認真實 set → VFD 主接觸器斷電
  8. Door Open：CPU 命令 → Door Operator 啟動 → vane skate roller 推 landing 鎖機構 → hook 翻起 → interlock NC 斷 → LED 紅閃 → cab door + landing door 同步打開
  9. Door Hold + Close Timer：門全開保持 4 秒 → 開始關門
  10. Door Close + Lock：cab door + landing door 同步關閉 → hook 重力下垂 → 勾住 lock body → interlock NC 閉合 → LED 變綠 → CPU 等下一個 call

5HVAC 空調系統 — Air Handling Unit

PUSD 辦公室採用單一 AHU（Air Handling Unit）+ 冰水盤管（CHW Coil）+ 全變頻送風機（VFD-VAV）的中央空調系統。室外新風經 F7 袋式濾網過濾、冰水盤管冷卻除濕、VFD 控制送風量、回風混和後再循環。控制以 BA Controller 為核心、依 CO₂ / 溫度 / 靜壓 PID 調節。
機構
5.1 AHU 主機模組（氣流順序）
  1. OA Damper（外氣風閥 DM）：馬達驅動葉片、開度 0–100%、決定引進新風比例
  2. F7 袋式濾網：過濾 PM2.5、空氣品質、初始壓損 80 Pa、終止壓損 250 Pa（換濾警示）
  3. 冰水盤管 CHW Coil：7°C 冰水進、12°C 出、銅管 + 鋁鰭片、冷量 25 RT
  4. 離心送風機 Centrifugal Fan：BLDC 馬達 + VFD 變頻、額定風量 6000 CMH、靜壓 500 Pa
  5. VFD 變頻器：調節送風機轉速、隨負荷變化、節能 30–50%
  6. 回風混和：從室內回風 + 新風混合、再循環
驅動
5.2 VFD 風機速度控制

送風機馬達為 BLDC、轉速由 VFD 透過改變頻率調節。靜壓感測器 PDT-107 即時量測管路靜壓、回授到 BA Controller、PID 計算後給 VFD 速度命令、實現定靜壓變風量（VAV）控制。

VFD 型式標準 ABB ACS580 或同級品
輸入380V 3φ
輸出頻率0–60 Hz 連續調節
控制週期10 ms PI loop
節能效果相較定速 30–50%
機構
5.3 Damper 模式切換

OA Damper 由 BA Controller 控制、依 CO₂ 濃度 + 室外溫度自動切換 3 種模式：

最小新風模式CO₂ < 800 ppm + 室外溫差 > 6°C → 新風比 20%（節能、減少冷負荷）
標準模式CO₂ 800–1000 ppm → 新風比 30% PID 調節
強制換氣模式CO₂ > 1200 ppm → 新風 100%（IAQ 警示）
Free Cooling室外溫度 < 18°C + 室內需冷 → 新風 100%、關冰水閥（免費冷氣）
時序
5.4 BA Controller 控制邏輯（PID + CO₂ + 安全聯鎖）

主要控制環路（4 條）

  • 送風溫度環：TT-105（送風溫度）回授 → PID → MV-01 冰水二通閥開度（PI、Kp=2.5, Ti=180s）
  • 靜壓環：PDT-107 → PID → VFD 速度命令（PI、Kp=0.8, Ti=60s）
  • 新風環：CO2-103 → PID → DM 外氣風閥開度（PI、Kp=1.5, Ti=240s、慢調避免震盪）
  • 濾網警報：ΔP-101 過濾器壓差 > 250 Pa → 警示 + 仍持續運轉

安全聯鎖

  1. SD-201 煙感觸發：立刻停送風機、關 OA damper、防止煙擴散
  2. VFD 故障：BA Controller 收到 fault → 關 OA damper、關冰水閥、警報
  3. FS-100 風速異常：送風機運轉中、風速 sensor 沒讀數 → 5 秒後判定皮帶斷或葉片卡 → 停機警報
